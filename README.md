# handmade-support-app

ハンドメイド作家の販売活動/ブランディングを支援するアプリ（ポートフォリオ）
在庫管理に加え、写真やタイトルの改善を通じて売上やブランド価値の向上をサポートする。

## ◇背景
ハンドメイド作品においては、写真が売上に大きく影響する。
一方で、既存サービスではこれらのノウハウはガイド記事として提供されることが多く、
出品時に活用されにくいと感じた。

## ◇特徴
・在庫管理機能  
・写真改善のためのガイド表示   

## ◇コンセプト
クリエイターの独創性を尊重しながら、販売活動とブランディングに必要な気づきを提供する
